# SimpleMP
A small SMP core for you to load up and play with your friends on!

# Commands
- `/sethome <name>`: Sets your home
- `/home <name>`: Teleports to your home
- `/delhome <name>`: Deletes the specified home
- `/homes`: Lists your homes
